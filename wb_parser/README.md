# Wildberries parser

Минимальный парсер каталога Wildberries по поисковому запросу с выгрузкой в XLSX.

## Что делает

- собирает каталог по запросу `пальто из натуральной шерсти`
- сохраняет полный каталог в `wb_parser/output/wildberries_catalog_full.xlsx`
- сохраняет отдельную выборку в `wb_parser/output/wildberries_catalog_filtered.xlsx`
- фильтрует товары по условиям:
  - рейтинг не ниже `4.5`
  - цена не выше `10000`
  - страна производства `Россия`

## Поля в XLSX

- Ссылка на товар
- Артикул
- Название
- Цена
- Описание
- Ссылки на изображения через запятую
- Все характеристики с сохранением структуры
- Название селлера
- Ссылка на селлера
- Размеры товара через запятую
- Остатки по товару (число)
- Рейтинг
- Количество отзывов

## Запуск

```bash
python3 -m pip install -r wb_parser/requirements.txt
python3 wb_parser/parse_wb_catalog.py
```


```bash
python3 wb_parser/parse_wb_catalog.py --query "пальто из натуральной шерсти" --output-dir wb_parser/output
```

## Примечание

Wildberries периодически ограничивает частоту запросов. В скрипте добавлены паузы и повторные попытки, а сбор остатков реализован как best-effort: если публичный endpoint остатков не отвечает, файл все равно будет собран, а поле остатков останется пустым.
